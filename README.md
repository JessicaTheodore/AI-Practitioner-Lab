Phish or Legit? is an educational Java game designed to teach users how to identify phishing emails versus legitimate messages.

Players are shown sample emails and must decide whether each one is phishing or legit. The game awards points for correct answers and explains key clues after each round to reinforce learning.

This project was developed as part of the Practitioner Lab Seminar on Gamified Social Engineering Training, demonstrating how interactive learning and gamification can improve cybersecurity awareness.
